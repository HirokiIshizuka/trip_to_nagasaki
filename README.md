# 長崎・雲仙 旅行ルートシミュレーター 🗺️

星野リゾート **界 雲仙** を拠点にした、レンタカー・1泊2日の長崎旅行プランを
地図上でシミュレーションできる Web アプリです。
食事・アクティビティのスポットを選び、回る順番を決めて、
**所要時間・移動距離・到着予定時刻**を確認できます。

## 使い方

1. このフォルダの `index.html` をブラウザで開くだけで動きます（ビルド不要）。
   - うまく動かない場合は簡易サーバー経由で開いてください：
     ```bash
     python3 -m http.server 8000
     # ブラウザで http://localhost:8000 を開く
     ```
2. 上部の **おすすめプラン** を選ぶと、1泊2日のモデルコースが読み込まれます。
3. 左の一覧やカテゴリ（🍜グルメ / 🏯歴史 / ⛰️自然 / ♨️温泉）からスポットを選び、
   **＋** ボタンで Day1 / Day2 のルートに追加します。
4. 右パネルの項目は **ドラッグで並べ替え** できます。
5. 各 Day の **出発時刻** を入れると、各スポットの到着予定時刻と移動時間が自動計算されます。
6. 作ったルートはブラウザに自動保存されます（次回開いたとき復元）。

## 特長

- 🗺️ Leaflet + OpenStreetMap 使用（**APIキー不要**でそのまま動く）
- 🚗 車での経路は OSRM で実際の道路に沿って計算（ネット接続時）。
  オフライン時は直線距離からの概算に自動フォールバック。
- 📍 長崎市〜雲仙〜小浜〜島原の主要スポットを収録。
- 💾 ルートはブラウザの localStorage に保存。

## 収録エリア・スポット例

- **長崎市**：グラバー園、大浦天主堂、平和公園、原爆資料館、出島、眼鏡橋、
  稲佐山夜景、新地中華街、四海樓（ちゃんぽん発祥）、ツル茶ん 他
- **雲仙**：雲仙地獄、仁田峠ロープウェイ、満明寺 他
- **小浜温泉**：ほっとふっと105（日本一の足湯）、小浜ちゃんぽん
- **島原**：島原城、武家屋敷、鯉の泳ぐまち、かんざらし、がまだすドーム

## 注意

- 各スポットの座標・所要時間は**目安**です。実際の出発前にカーナビ等で確認してください。
- スポットの追加・編集は `js/data.js` の `POIS` 配列を編集すると行えます。

## ファイル構成

```

```
